Investigación IEIE - Evaluación WCAG

Análisis detallado de accesibilidad web para el Instituto Europeo de Inteligencias Eficientes

Institución Evaluada

Instituto Europeo de Inteligencias Eficientes (IEIE)

1. Perceptible

Parcial
  • ✅ Imágenes con atributos alt
  • ❌ Texto alternativo no descriptivo (ej: "academia-de-emociones-ieie")

2. Operable

Parcial
  • ✅ Enlace "Saltar al contenido" presente
  • ⚠️ Posible estructura jerárquica de encabezados incorrecta

3. Comprensible

Parcial
  • ❌ Falta atributo lang en <html>
  • ⚠️ Encabezados con texto no semántico (ej: fecha en título)

4. Robusto

No Conforme
  • ❌ Ausencia de roles ARIA o elementos semánticos modernos

Resumen de Conformidad

Principio WCAG Estado Observaciones
Perceptible Parcial Alt text no descriptivo
Operable Parcial Posible jerarquía de encabezados incorrecta
Comprensible Parcial Sin atributo lang y encabezados no semánticos
Robusto No Conforme Sin ARIA o HTML5 semántico

Recomendaciones de Mejora

  1. Agregar lang="es" al <html> para identificar el idioma principal
  2. Mejorar texto alternativo descriptivo (ej: "Academia de emociones - IEIE")
  3. Corregir jerarquía de encabezados siguiendo estructura H1 > H2 > H3
  4. Implementar elementos semánticos <nav>, <main> y <footer>
  5. Usar atributos ARIA para roles interactivos y dinámicos
  6. Verificar contraste de colores con herramientas especializadas como WAVE
